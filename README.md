# Quantium Data Processing Task

## 📌 Objective
Analyze sales data to determine whether sales increased after the Pink Morsel price change on 15th January 2021.

## 📂 Data Processing Steps
- Combined multiple CSV files into a single dataset
- Filtered data for "pink morsel" products
- Created a new "Sales" column (quantity × price)
- Selected required columns: Sales, Date, Region

## 📊 Analysis
Calculated total sales:
- Before 15 Jan 2021
- After 15 Jan 2021

## 🧾 Results
Sales were higher after 15 January 2021.
